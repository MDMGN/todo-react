# Ejercicios para modificar y eliminar nuestras tareas de la lista

- **TodoItem** recibe las siguientes propiedades
    - id
    - title
    - completed

- Modificar la función **addTodo** para que pueda recibir estos argumentos para  guardarlos en el **estado** y **localStorage**

- Modificar **TodoItem**:
    - Agregar un botón para editar la tarea.
    - Agregar un botón para eliminar la tarea.

- Agregar el código necesario para que nuestra función **updateTodo** pueda modificar una tarea recibiendo una tarea ->**id,title,completed** 

- Agregar el código necesario para que nuestra función **removeTodo** pueda eliminar una tarea recibiendo su **id**.
